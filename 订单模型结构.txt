================================================================================
                                关系说明
================================================================================

一对一关系 (1:1):
  - Order ↔ OrderReceiver (订单收货人)
  - Order ↔ OrderContact (订单联系人 - 每个订单有买家和卖家两个联系人)
  - Order ↔ OrderRateInfo (订单评价信息)
  - Order ↔ OrderBizInfo (订单业务信息)
  - Order ↔ OrderLogistics (订单物流信息)
  - Store ↔ PlatformApiConfig (平台API配置)
  - Store ↔ StoreDataConfig (店铺数据配置)

一对多关系 (1:N):
  - Order → OrderItem (订单商品)
  - Order → OrderStep (订单步骤)
  - Order → OrderTradeTerm (订单交易条款)
  - OrderLogistics → OrderLogisticsItem (订单物流明细)
  - Store → Order (订单)
  - Store → DataPullTask (数据拉取任务)
  - Store → StoreData (店铺数据记录)

多对多关系 (M:N):
  - Store ↔ User (店铺管理员)
