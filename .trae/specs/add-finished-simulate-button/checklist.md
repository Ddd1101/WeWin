# Checklist

- [x] 成品详情页（product_type == "finished"）显示"克价模拟"按钮
- [x] 非成品类型（bead / accessory）详情页不显示"克价模拟"按钮
- [x] 点击"克价模拟"按钮打开模拟弹窗
- [x] 弹窗展示串珠明细：名称、数量、克重、原克价、新克价（可编辑）、原小计、新小计
- [x] 弹窗展示配件明细：名称、数量、原单价、新单价（可编辑）、原小计、新小计
- [x] 串珠新小计 = 新克价 × 克重 × 数量
- [x] 配件新小计 = 新单价 × 数量
- [x] 无串珠和配件时显示"无明细"
- [x] 弹窗底部展示：原成本、新成本、成本变动、原利润、新利润、原利润率、新利润率
- [x] 原成本 = Σ(串珠 bead_cost_price × quantity) + Σ(配件 accessory_cost_price × quantity) + labor_cost + elastic_cost
- [x] 新成本 = Σ(新克价 × bead_weight × quantity) + Σ(新单价 × quantity) + labor_cost + elastic_cost
- [x] 新成本 > 原成本时，新成本和成本变动显示红色；否则绿色
- [x] 新利润率 ≥ 30% 显示绿色；15%-30% 显示橙色；< 15% 显示红色
- [x] 新利润 < 0 显示红色；≥ 0 显示绿色
- [x] 修改新克价/新单价时，汇总指标实时更新
- [x] "重置"按钮将所有新克价/新单价恢复为原值
- [x] "关闭"按钮关闭弹窗，不发送后端请求
- [x] Gradle 编译通过，无编译错误
- [x] 真机验证：模拟按钮显示、弹窗交互、计算正确、重置/关闭功能正常
