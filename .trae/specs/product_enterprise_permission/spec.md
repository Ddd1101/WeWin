# 商品管理企业权限控制 - 产品需求文档

## 概述
- **摘要**: 实现商品管理模块的企业级权限控制，确保企业下属用户创建的商品仅该企业员工可见
- **目的**: 保护企业数据安全，防止不同企业间的商品数据泄露
- **目标用户**: 企业负责人、企业管理员、企业普通用户、网站管理员、网站超级管理员

## 目标
- ✅ 企业用户只能查看本企业的商品
- ✅ 企业用户只能操作本企业的商品
- ✅ 网站管理员和超级管理员可查看所有企业商品
- ✅ 权限控制覆盖商品的增删改查全流程

## 非目标（超出范围）
- 不实现企业内细粒度的角色权限（如部门级、用户级权限）
- 不修改商品的数据模型结构
- 不涉及前端UI的大改造

## 背景与上下文

### 现有系统架构
- **数据模型**:
  - `Product` 模型已有 `company` 外键关联企业
  - `User` 模型已有 `company` 外键关联企业
  - `UserType` 枚举定义了用户角色（SUPER_ADMIN、SITE_ADMIN、ENTERPRISE_LEADER、ENTERPRISE_ADMIN、ENTERPRISE_USER、TEMPORARY）

- **现有权限实现分析**:
  通过分析代码发现，大部分权限控制逻辑已经实现！包括：
  - `get_products` - 商品列表已按企业过滤
  - `get_product_detail` - 商品详情已有权限检查
  - `create_product` - 创建商品已校验企业权限
  - `update_product` - 更新商品已校验企业权限
  - `delete_product` - 删除商品已校验企业权限
  - `get_beads` - 串珠列表已按企业过滤
  - `get_accessories` - 配件列表已按企业过滤

### 技术限制
- 使用 Django 作为后端框架
- 使用 JWT 进行身份认证
- 使用 PostgreSQL/MySQL 数据库

## 功能需求

### FR-1: 商品列表权限控制
- 网站超级管理员和网站管理员：可查看所有企业商品
- 企业负责人、企业管理员、企业普通用户：仅可查看本企业商品
- 临时用户：无查看商品权限

### FR-2: 商品详情权限控制
- 网站超级管理员和网站管理员：可查看所有企业商品详情
- 企业负责人、企业管理员、企业普通用户：仅可查看本企业商品详情
- 临时用户：无查看商品详情权限

### FR-3: 商品创建权限控制
- 网站超级管理员和网站管理员：可创建任意企业商品
- 企业负责人、企业管理员：可创建本企业商品
- 企业普通用户：无创建商品权限
- 临时用户：无创建商品权限

### FR-4: 商品更新权限控制
- 网站超级管理员和网站管理员：可更新任意企业商品
- 企业负责人、企业管理员：仅可更新本企业商品
- 企业普通用户：无更新商品权限
- 临时用户：无更新商品权限

### FR-5: 商品删除权限控制
- 网站超级管理员和网站管理员：可删除任意企业商品
- 企业负责人：仅可删除本企业商品
- 企业管理员、企业普通用户：无删除商品权限
- 临时用户：无删除商品权限

### FR-6: 串珠和配件列表权限控制
- 网站超级管理员和网站管理员：可查看所有串珠和配件
- 企业负责人、企业管理员、企业普通用户：仅可查看本企业串珠和配件
- 临时用户：无查看权限

## 验收标准

### AC-1: 企业用户商品列表查看
- **给定**: 用户已登录且属于企业A，企业B有商品
- **当**: 用户调用商品列表接口
- **则**: 仅返回企业A的商品，不包含企业B的商品
- **验证方式**: 程序验证（单元测试 + 集成测试）

### AC-2: 企业用户商品详情查看
- **给定**: 用户已登录且属于企业A，商品1属于企业A，商品2属于企业B
- **当**: 用户请求查看商品2详情
- **则**: 返回权限不足错误
- **验证方式**: 程序验证

### AC-3: 企业用户创建商品
- **给定**: 用户已登录且属于企业A，用户角色为企业管理员
- **当**: 用户尝试为企业B创建商品
- **则**: 创建失败，返回权限不足
- **验证方式**: 程序验证

### AC-4: 企业用户更新商品
- **给定**: 用户已登录且属于企业A，商品属于企业B
- **当**: 用户尝试更新该商品
- **则**: 更新失败，返回权限不足
- **验证方式**: 程序验证

### AC-5: 企业用户删除商品
- **给定**: 用户已登录且属于企业A，商品属于企业B，用户角色为企业负责人
- **当**: 用户尝试删除该商品
- **则**: 删除失败，返回权限不足
- **验证方式**: 程序验证

### AC-6: 网站管理员查看所有商品
- **给定**: 用户已登录且角色为网站管理员
- **当**: 用户调用商品列表接口
- **则**: 返回所有企业的商品
- **验证方式**: 程序验证
