# 商品管理企业权限控制 - 验证清单

## 代码审核检查点
- [ ] 检查 get_products 函数是否正确按企业过滤商品
- [ ] 检查 get_product_detail 函数是否有正确的权限检查
- [ ] 检查 create_product 函数是否限制了企业用户只能创建本企业商品
- [ ] 检查 update_product 函数是否限制了企业用户只能更新本企业商品
- [ ] 检查 delete_product 函数是否限制了企业用户只能删除本企业商品
- [ ] 检查 get_beads 函数是否正确按企业过滤串珠
- [ ] 检查 get_accessories 函数是否正确按企业过滤配件
- [ ] 检查权限检查是否覆盖所有用户角色（包括临时用户）
- [ ] 检查代码风格是否与现有代码一致
- [ ] 检查是否有重复的权限检查逻辑可以提取

## 功能测试检查点

### 测试数据准备
- [ ] 创建企业 A 和企业 B
- [ ] 创建各角色用户（网站超级管理员、网站管理员、企业负责人、企业管理员、企业普通用户、临时用户）
- [ ] 为企业 A 和企业 B 各创建一些测试商品

### 商品列表权限 (AC-1)
- [ ] 企业普通用户只能看到本企业商品
- [ ] 企业管理员只能看到本企业商品
- [ ] 企业负责人只能看到本企业商品
- [ ] 网站管理员能看到所有企业商品
- [ ] 网站超级管理员能看到所有企业商品
- [ ] 临时用户无查看商品权限

### 商品详情权限 (AC-2)
- [ ] 企业普通用户只能查看本企业商品详情
- [ ] 企业用户尝试查看其他企业商品详情应返回权限错误
- [ ] 网站管理员能查看所有企业商品详情

### 商品创建权限 (AC-3)
- [ ] 企业负责人能创建本企业商品
- [ ] 企业管理员能创建本企业商品
- [ ] 企业普通用户不能创建商品
- [ ] 临时用户不能创建商品
- [ ] 企业用户尝试为其他企业创建商品应失败

### 商品更新权限 (AC-4)
- [ ] 企业负责人能更新本企业商品
- [ ] 企业管理员能更新本企业商品
- [ ] 企业普通用户不能更新商品
- [ ] 企业用户尝试更新其他企业商品应失败

### 商品删除权限 (AC-5)
- [ ] 企业负责人能删除本企业商品
- [ ] 企业管理员不能删除商品
- [ ] 企业普通用户不能删除商品
- [ ] 企业用户尝试删除其他企业商品应失败

### 网站管理员全权限 (AC-6)
- [ ] 网站管理员能查看所有企业商品
- [ ] 网站管理员能创建任意企业商品
- [ ] 网站管理员能更新任意企业商品
- [ ] 网站管理员能删除任意企业商品

### 串珠和配件权限
- [ ] 企业用户只能看到本企业串珠
- [ ] 企业用户只能看到本企业配件
- [ ] 网站管理员能看到所有串珠和配件

## 边界情况测试
- [ ] 用户无所属企业时的处理
- [ ] 用户类型变更时的权限变更
- [ ] 商品所属企业变更时的权限影响
- [ ] 并发操作时的权限一致性

## 性能检查点
- [ ] 权限检查响应时间 < 50ms
- [ ] 大数据量情况下查询性能正常
