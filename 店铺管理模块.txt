1. 我想增加一个不同电商平台的店铺后台数据管理平台
2. 涉及的电商平台包括：1688、拼多多、抖音电商、淘宝等
3. 每个电商平台可能涉及多个不同的店铺
4. 不同的店铺设计不同的领域，有童装、水晶手串、耳钉耳饰等
5. 企业负责人和企业管理员可以创建店铺，也可以配置哪些企业下的人员可以进行管理
6. 每一个店铺都有对应的数据管理页面
7. 每个企业账户可以对其管理的店铺进行统一管理


## 后端实现
### 1. 数据模型 (models.py)
- Platform : 电商平台枚举（淘宝、天猫、京东、拼多多、抖音电商、快手、1688、小红书、微信小店等）
- Category : 店铺品类枚举（童装、水晶手串、耳钉耳饰、项链、戒指、箱包、鞋类等）
- Store : 店铺模型，包含以下字段：
  - 店铺名称、电商平台、店铺品类
  - 所属企业、店铺管理员（多对多）
  - 店铺描述、店铺链接、联系人信息
  - 状态、创建人、创建时间等
### 2. API 接口 (views.py)
- GET /api/account/platforms/ - 获取所有电商平台列表
- GET /api/account/categories/ - 获取所有店铺品类列表
- GET /api/account/stores/ - 获取店铺列表（根据用户权限过滤）
- POST /api/account/stores/create/ - 创建店铺
- PUT /api/account/stores/<id>/update/ - 更新店铺
- DELETE /api/account/stores/<id>/delete/ - 删除店铺
### 3. 权限控制
- 超级管理员/网站管理员 : 可以查看和管理所有店铺
- 企业负责人/企业管理员 : 可以查看和管理本企业的店铺
- 企业用户 : 只能查看自己管理的店铺
## 前端实现
### 1. API 封装 (api/index.js)
添加了店铺管理相关的 API 函数

### 2. 店铺管理页面 (Stores.vue)
- 店铺列表展示 : 表格展示店铺信息，包含平台标签和状态标签
- 创建店铺 : 弹窗表单，支持选择平台、品类、管理员等
- 编辑店铺 : 修改店铺信息，支持启用/禁用店铺
- 删除店铺 : 带确认对话框的删除功能
- 平台标签 : 不同平台显示不同颜色的标签
